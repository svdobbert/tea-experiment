# tea-experiment

### Some code to analyse samples from an experiment involving environmental impacts on tea decomposition times.

[![Build Status](https://github.com/svdobbert/tea-experiment/workflows/CI/badge.svg)](https://github.com/svdobbert/tea-experiment/actions)

# <span style="color:green"> **About** </span> 

This Repository contains some statistical models to analyse climate data and tea decomposition times in alpine regions in Central Norway.

To get started with julia and VS Code: https://www.julia-vscode.org/docs/dev/gettingstarted/
